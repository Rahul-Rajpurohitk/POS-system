# RPOS - React Native POS App
### Installation
##### Run server source code
-   Run "yarn install" to install all dependencies
-   Run "yarn start" to run the source code
##### Run app source code
-   Run "yarn install" to install all dependencies
-   cd ios folder and run "pod install"
-   Run "react-native run-ios" or "react-native run-android" to run source code

### Support
If you have any issues when run the project, you can email to admin@mposs.io. We will be ready for supporting to check issues remotely.
